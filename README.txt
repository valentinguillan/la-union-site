
# La Unión — Sitio institucional (estático)
Estructura separada por lenguaje:
- `index.html`
- `css/style.css`
- `js/main.js`
- `assets/` (logo, favicon y OG image)
- `img/products/` (imágenes de productos)

## Cómo usar
1. Abrí `index.html` en tu navegador (funciona localmente).
2. Subí toda la carpeta a Netlify o GitHub Pages.
3. Reemplazá assets e imágenes por los definitivos.

## Notas
- El formulario usa Formspree: actualiza `action` con tu endpoint.
- El mapa es un embed por query de Google Maps.
- Tipografías: usa Futura PT si la tenés con licencia; si no, quedará el fallback del stack del sistema.
